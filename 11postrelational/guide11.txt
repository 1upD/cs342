1.a.	Explain the tiers discussed in the article. On which does J2EE focus? On which to the Oracle database focus?
	Client		-	Applications that access a Java EE server and are located on a different machine from the server.
	Web			-	Components that handle the interaction between clients and the business tier.
	Business	-	Components that provide business logic for application.
	Enterprise	-	Database servers, enterprise resource planning, data sources like mainframes. Typically located on a separate machine.

1.b.	Explain the nature of a J2EE container. What sort of containers are there?
	Containers interface between EE components and low level platforms. Web containers interface between web components and the web server. EJB containers interface between business tier logic and the Java EE server. Application Client containers interface between Java EE clients and the Java EE server.
